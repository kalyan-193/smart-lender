# smart-lender
repo link : https://github.com/kalyan-193/SmartLender
website link:https://smartlender-yhv0.onrender.com/
Smart Lender AI is an intelligent loan approval prediction system built using Machine Learning, Flask, and Python. The application analyzes applicant details and predicts loan eligibility using a trained Random Forest model, helping financial institutions make faster and data-driven lending decisions.
